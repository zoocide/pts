NAME

Plugins::Base::Util - A selection of useful utility functions to write your own plugin for the PTS system.

FUNCTIONS

get_package_dir($package)

Return the directory containing the package .pm file. By default $package is the caller's __PACKAGE__. E.g., get_package_dir() returns /a/lib/path for the path /a/lib/path/Plugins/Your/Plugin/Name.pm.
