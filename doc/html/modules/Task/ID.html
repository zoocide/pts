DESCRIPTION

METHODS

Task::ID->new($s)

It returns the object constructed from the $s string.

reset($s)

It replaces the object content to correspond the specified $s string.

id

The string of form 'a/path/task_name:arg1=value1,arg2=e1 e2'.

short_id

The string of form 'a/path/task_name'.

basename

The string of form 'task_name'.

dirs

It returns a list ('a', 'path') for the task path 'a/path/task_name.conf'.

args

It returns a hash with the task arguments (e.g. ('gr_1' => {arg1 => ['elm 1',...],...},...)).

args_hash

Similar to the args method, but returns a hashref instead of a hash (e.g. {'gr_1' => {arg1 => ['elm 1',...],...},...}).

args_str

It returns the argument part of the ID string (e.g. 'arg1=value1,arg2=e1 e2').

has_args

It returns true if the task has any arguments.

has_arg($group, $var)

It returns true if the argument is specified for the task.

FUNCTIONS

arg2str($gr, $var, @value)

It makes an argument-like string with the specified vairiable and value. The "${gr}::" prefix will be ommited, if the $gr is undef. The "${gr}::${var}=" part will be ommited, if the $var is undef.
